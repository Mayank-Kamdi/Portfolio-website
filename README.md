# Mayank Kamdi — Premium Portfolio

A high-end, cinematic portfolio website built with React, Framer Motion, and WebGL.

## 🚀 Key Features
- **Typewriter Preloader**: A bespoke character-reveal loading experience.
- **WebGL Waterfall Hero**: A high-fidelity liquid displacement portrait with interactive mouse-parallax.
- **Architectural Layout**: Inspired by premium designer portfolios, featuring minimal typography and grid overlays.
- **Dynamic UI**: Smooth entrance animations and live DEL/IST clock.

## 🛠️ Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion + custom GLSL Shaders
- **Icons**: Lucide React

## 📦 Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
